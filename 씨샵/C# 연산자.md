## 연산자
산술 연산자: +, -, *, /, %        
증감 연산자: ++, --    
문자열 결합: string result = "문자열1" + "문자열2" 형태  
관계 연산자: <, >, ==, !=, <=, >=이고 평가 후 반환값은 True or False  
조건 연산자: 조건식 ? b : c이고 반환값은 조건식이 참이면 b 거짓이면 c  
null 조건부 연산자: ?., ?[]  
ex) a?.b , a?[b]
a가 null이면 null 반환, null이 아니면 b 반환  
null 병합 연산자: ??  
ex) a ?? b a가 null이 아니면 a 반환 null이면 b 반환  
논리 연산자: &&논리곱AND, ||논리합OR, !부정NOT이고 반환값은 True of False   
비트 연산자: <<, >>, &, |, ^, ~  
ex) a << 값 => a의 비트를 왼쪽으로 값만큼 이동  
a & b => a: 00001111, b: 00111111이라면 각 자릿수를 비교해 00001111 반환  
할당 연산자: =, +=, -=, *=, /=, %=, &=, |=, ^=, <<=, >>=이고 왼쪽 피연산자에 값 저장
연산자의 우선순위: 
1. 후위, null 조건부 연산자 2. 전위 연산자 3. 산술 연산자 4. 시프트 연산자 5. 관계 연산자  

느낀점: 개념을 정리하는 단계라 복습하는 느낌이었다.  
c를 배울 때 모호했던 비트연산자는 이제야 이해가 된다    
전위, 후위의 반환값과 포인터값에 대해 예전에 정리했던 내용을 기록하고 싶은데  
연산자 우선순위를 정리하다 보니 약속시간이 되어 나가야 한다...
